# Student Management System

A Django web application to manage student records. Users can add, view, edit and delete students.

## Features
- Add new student details
- View student list
- Edit and Delete student records  
- Django Admin panel
- SQLite database

## Tech Stack
- **Backend**: Python, Django 5.2
- **Frontend**: HTML, CSS
- **Database**: SQLite
